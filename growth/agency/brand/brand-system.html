03

Wordmark & layout

wordmark

lowercase tsukumo, Archivo 800, acid square mark before the word (height ≈ cap height). Clear-space = one square on all sides. Min 96px / 20mm, else square-only.

layout

12-col, content on odd ranges (hang off the grid). Hairline rules + margin index numbers. Hard corners (radius 0). Negative space is a material. Motion: heavy ease-out, masked type reveals, honor reduced-motion.

04

Do / don't

  • ✅ one acid accent per view
  • ✅ oversized type, raw asymmetric grid
  • ✅ mono labels + index numbers
  • ✅ concrete texture, hard directional light
  • ✅ real words, qualitative proof
  • ✕ more than one accent in view
  • ✕ centered safe layouts, rounded friendly corners
  • ✕ gradients / blobs / stock photos / glowing brains
  • ✕ fabricated numbers, logos, testimonials
  • ✕ the name "Synergix" on any public surface
