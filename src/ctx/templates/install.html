Install the ctx ambient hook.

1. Create ~/.claude/hooks/ctx-ambient.sh with the content fetched from
   https://ctx.prod.synergix.ch/hooks/ctx-ambient.sh
2. chmod +x it.
3. Read ~/.claude/settings.json and add this to hooks.PostToolUse (merge with
   existing entries — don't overwrite):

     {
       "matcher": "Read",
       "hooks": [{
         "type": "command",
         "command": "/home/<me>/.claude/hooks/ctx-ambient.sh",
         "timeout": 5
       }]
     }

   Replace <me> with $HOME's basename.
4. Tell me to restart Claude Code.
